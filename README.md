# Enhanced Traceroute

Simple wrapper over the system traceroute command, tagging the hops with the ASN and geolocation.
For privacy reasons, the ASN and geolocation are queried from local databases only, rather than public APIs.

__Generated by Claude 3.5 Sonnet__

## Setup

### Install dependencies

```bash
pip install geoip2
```

### Download databases

Please sign up for a free account at [MaxMind](https://www.maxmind.com/en/geolite2/signup) and download `GeoLite2-ASN.mmdb` and `GeoLite2-City.mmdb`.

## Usage

```bash
python -m enhanced_traceroute.traceroute <target>
```
